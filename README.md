# 🗳️ Online Voting System (JavaFX + Maven)

## 📌 Overview

The **Online Voting System** is a desktop application developed using **JavaFX** and **Maven**. It enables secure and efficient digital voting with separate access for administrators and voters. The system ensures that each voter can cast their vote only once.

---

## 🎯 Features

### 👨‍💼 Admin Panel

* Secure admin login
* Add, update, and delete candidates
* View voting results
* Manage voter records

### 🧑‍🤝‍🧑 Voter Panel

* Secure voter login
* Vote for preferred candidates
* One-person-one-vote restriction
* Simple and user-friendly interface

---

## 🛠️ Technologies Used

* Java
* JavaFX
* Maven
* File Handling (.dat files)

---

## 💾 Data Storage

* `candidates.dat` → Stores candidate information
* `voters.dat` → Stores voter data and voting status

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/Online-Voting-System.git
   ```
2. Open the project in an IDE (IntelliJ / NetBeans / Eclipse)
3. Build using Maven
4. Run the JavaFX application
